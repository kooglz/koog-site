# Koog Site

A minimalist portfolio site built with React + Vite + TailwindCSS.

## Project Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
